Arduino101 Scratch Extension with machine learning

A easy way to let kids know what is machine learning

Download as .zip Download as .tar.gz View on GitHub

支持机器学习功能的Arduino 101 Scratch扩展

本扩展源码编译需要General Vision的CurieNeurons free library支持。

使用方法

烧写firmware

下载已编译的firmware及烧写工具，

解压

如果未安装驱动，请先运行tools\driver_install.bat，安装arduino 101驱动

运行upload.bat烧写firmware。

启动scratch并加载扩展

http://scratchx.org/?url=http://xmu-maker.github.io/scratch-arduino101-neurons-ext/arduino101-neurons-scratch-extension.js
